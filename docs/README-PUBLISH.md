# Publicação no GitHub Pages

Este diretório já está pronto para publicação como página wiki.

## Passos

1. Suba o repositório para o GitHub.
2. Vá em **Settings -> Pages**.
3. Em **Build and deployment**, selecione:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (ou `master`)
   - **Folder:** `/docs`
4. Salve e aguarde o deploy.

## Arquivo principal

- `docs/index.md` (página wiki principal).

## Observação comercial

Antes de vender conteúdo derivado (Amazon/KDP), valide licenças de imagens e marcas de terceiros citadas no conteúdo.
